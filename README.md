# ionic-peliculas

 - Ir a la terminal y ejecutar el comando **npm install** dentro del directorio del proyecto
 - Ejecutar **ionic serve** para poder visualizar el proyecto en la web
 - En caso de que prefiera probarlo en un móvil android, abrir en Android Studio la carpeta android del directorio root del proyecto y correr la aplicación en un emulador o móvil real.
 - En caso de que prefiera probarlo en un móvil IOS, abrir en Xcode carpeta ios del directorio root del proyecto y correr la aplicación en un emulador o móvil real.
 
Aplicación de peliculas utilizando recursos de la API TheMovieDB y el Storage
